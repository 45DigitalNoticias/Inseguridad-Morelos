Versiones publicadas

Cada vez que el SESNSP publica un nuevo corte de incidencia delictiva, actualizamos el dashboard. Las versiones anteriores se preservan tal como fueron publicadas — sin modificación retroactiva — para que cualquier lector pueda verificar qué decían y cuándo.

¿Por qué guardamos versiones anteriores?

Los datos de seguridad pública pueden cambiar entre cortes: el SESNSP a veces revisa cifras pasadas cuando llegan reportes tardíos. Para que ninguna afirmación de este dashboard pueda ser cuestionada como "cambio silencioso de cifras", mantenemos públicas todas las versiones publicadas.

Si una nota periodística cita el dashboard hoy, esa cita seguirá siendo verificable mañana. Si SESNSP modifica retroactivamente alguna cifra, el archivo histórico documenta lo que decía la información oficial cuando publicamos. La trazabilidad es parte del producto.
